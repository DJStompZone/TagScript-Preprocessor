TagScript Preprocessor

If you're familiar with Sass/SCSS for web development, you know how convenient a preprocessor is.
It allows you to type less or have a more intuitive way to code than what you normally have to do.

Pro Tip: Press Ctrl + , to process once, and Ctrl + * to toggle real-time processing.
